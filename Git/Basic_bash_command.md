# BASH COMMAND

| 명령어                                                      | 설명        |
| ----------------------------------------------------------- | ----------- |
| `cd  <폴더 이름>`                                           | 폴더 열기   |
| `mkdir <폴더 이름>`                                         | 폴더 생성   |
| `touch <파일 이름>.py <파일 이름>.md`                       | 파일 생성   |
| `mv <현재 폴더 이름 >/<파일 or 폴더 이름> <옮길 폴더 이름>` | 파일 옮기기 |
|                                                             |             |
